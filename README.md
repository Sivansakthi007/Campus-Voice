🎓 Campus Voice

AI‑Powered Student Complaint Management System

Campus Voice is a modern, AI‑driven platform designed for educational institutions to efficiently collect, analyze, track, and resolve student grievances. It replaces traditional paper‑based complaint systems with an intelligent, transparent, and role‑based digital solution.


---

📌 Overview

Educational institutions often struggle with slow, non‑transparent, and inefficient complaint handling processes. Campus Voice bridges the communication gap between students and administration by providing:

24/7 accessible complaint submission

AI‑powered analysis and prioritization

Real‑time tracking and accountability

Role‑based dashboards for all stakeholders



---

🚀 Key Features

1. Intelligent Complaint Submission

Text & Voice input (Voice‑to‑Text support)

AI‑based automatic categorization

Anonymous complaint option

Duplicate complaint detection


2. AI‑Powered Analysis

Sentiment Analysis (Positive, Negative, Angry, Urgent)

Automatic Priority Assignment (High / Medium / Low)

Profanity & Foul Language Detection

Multi‑language support (English, Tamil, Hinglish)


3. Role‑Based Dashboard System

Role	Capabilities

Student	Submit complaints, track status, view responses
Staff	View assigned complaints, respond, update status
HOD	Department oversight, assign staff, review all
Principal	Institution‑wide view, analytics, oversight
Admin	User management, system configuration, full access


4. Real‑Time Complaint Tracking

Status Timeline: Submitted → Reviewed → In Progress → Resolved / Rejected

Assignment tracking

Complete response history


5. Analytics & Reporting

Complaint overview dashboard

Category‑wise breakdown

Staff performance metrics

Trend analysis for recurring issues


6. Additional Features

Secure JWT‑based authentication

Encrypted password storage

Profile management

Responsive design (Desktop / Tablet / Mobile)

Dark & Light theme support

Complaint support voting
